# 카카오맵 API 설정 가이드

## 문제: 지도가 로드되지 않음

콘솔 에러: "카카오맵 스크립트 로드 실패"

## 해결 방법

### 1. 카카오 개발자 콘솔 접속
https://developers.kakao.com/

### 2. 내 애플리케이션 선택
- 로그인 후 내 애플리케이션 메뉴 클릭
- 해당 앱 선택

### 3. 플랫폼 설정
- 좌측 메뉴에서 **"앱 설정" > "플랫폼"** 클릭
- **"Web 플랫폼 등록"** 클릭

### 4. 사이트 도메인 등록
다음 도메인들을 모두 등록하세요:

**개발 환경:**
- `http://localhost:5173`
- `http://localhost:5174`
- `http://localhost:3000`
- `http://127.0.0.1:5173`

**프로덕션 환경:**
- `http://146.56.129.230:3000`
- `http://146.56.129.230` (포트 없이)

**참고:**
- 각 도메인을 개별적으로 추가해야 합니다
- 포트 번호가 있는 경우와 없는 경우 모두 등록하는 것이 좋습니다
- 저장 후 몇 분 정도 기다려야 적용됩니다

### 5. JavaScript 키 확인
- **"앱 키"** 메뉴에서 **JavaScript 키**를 확인하세요
- REST API 키가 아닌 **JavaScript 키**를 사용해야 합니다
- `.env` 파일의 `VITE_KAKAO_MAP_API_KEY`에 JavaScript 키를 입력하세요

### 6. API 사용 설정
- **"제품 설정" > "카카오 로그인"** 또는 **"제품 설정" > "카카오맵"** 메뉴에서
- 필요한 API를 활성화하세요

## 확인 방법

브라우저 콘솔에서 다음 메시지가 보이면 성공:
- ✅ "카카오맵 스크립트 로드 성공"
- ✅ "카카오맵 API 키가 설정되었습니다."

실패 시:
- ❌ "카카오맵 스크립트 로드 실패"
- ❌ 현재 도메인이 플랫폼에 등록되지 않았습니다

## 주의사항

1. **localhost는 개발용**: 프로덕션에서는 실제 도메인을 등록해야 합니다
2. **포트 번호 포함**: `localhost:5173`과 `localhost`는 다른 도메인으로 인식됩니다
3. **적용 시간**: 도메인 추가 후 즉시 적용되지 않을 수 있으니 몇 분 기다려보세요
